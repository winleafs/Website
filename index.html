Winleafs

The Open Source Windows desktop application for the Nanoleaf Lights
Get Started


Optimized for multiple devices

Winleafs has been build to support multiple devices in an easy and understandable way. Each device will act for itself, meaning if one device goes down, your schedule and effects will continue to work on your other device.

Screen mirror on anything

The screen mirror features is optimized to use as little resources as possible. Using a combination of a custom written algorithm and making smart use of the Windows APIs, we make screen mirror available on any application.

With the trigger system build into Winleafs, you can switch to the screen mirror effect whenever a specific program launches. Whether you watch movies in VLC or go for a game of Rocket League, Winleafs can automatically enable the effects for you.

Completely open source.

We believe in the open source community and transparency for our product. Winleafs has been open source from the moment it was founded, all side projects that have spawned out of Winleafs like this website and JsonMigrator.
